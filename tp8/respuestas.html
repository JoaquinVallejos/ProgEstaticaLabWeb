Ejercicio 2c

Las 3 formas de incluir código JavaScript son las siguientes:

Inline: Esta forma de incluir el código es directa al archivo HTML, mediante <.script> en el head o en el body, podria usarse en casos donde el script es muy simple o para una sola página.

Externo: Esta forma de incluir el código es mediante un archivo externo con extensión .js con <.script src="./ejemplo.js"> también en el head o en el body. Se podria utilizar cuando se necesita el mismo script en varias páginas.

JavaScript en elementos HTML: Esta forma se trata de agregar el codigo JavaScript directamente a los atributos de eventos de los elementos HTML, también podria usarse para acciones pequeñas o simples.
